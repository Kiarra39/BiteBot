#  BiteBot: AI Recipe Generator

BiteBot is a smart and simple AI-powered web app that helps you generate delicious recipes using ingredients you already have! Just type in whatâ€™s in your kitchen (like "rice, tomato, onion") and BiteBot will whip up a personalized recipe for you in seconds.

##  Features

-  AI-generated recipes using the Google Generative AI API
-  Ingredient-based recipe suggestions
-  Modern UI built with React and Tailwind CSS
-  Hosted on Render (backend) and Vercel (frontend)

##  Tech Stack

- **Frontend:** React, Tailwind CSS, Axios
- **Backend:** Node.js, Express, dotenv, Google Generative AI
- **Hosting:** Vercel (frontend) + Render (backend)
